This application needs steamVR installed and a HTC Vive attached to the computer.
For further information please visit http://store.steampowered.com/app/358040/HTC_Vive/

In order to change the data set of the visualization please replace ".\ScatterVR1.0_Data\StreamingAssets\ScatterplotData\points.csv".


Controls:
To move or rotate the scatter plot around, hold down the trigger button (back of the Vive controller) of one controller.
The scatter plot will now mimic any move made by the controller.
In order to change the size of the visualization, hold down the trigger button of both controllers and change the distance between the controllers.
